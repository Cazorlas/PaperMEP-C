PaperMEP is a free Autodesk Revit add-in,
written in C#, providing a suite of handy MEP tools.
Every command comes with a built-in video tutorial
available directly in the tooltip.

üì• Installation
1. Run PaperMEP.1.0.0.exe
2. The installer will automatically copy the bundle into:

3. If an older version is detected, the installer will
   automatically uninstall it first.
